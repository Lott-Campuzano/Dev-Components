Elementos Obsoletos
<!DOCTYPE>

Declara el tipo de documento y la versión de HTML que se está utilizando.

Sintaxis:
<!DOCTYPE html>
Atributos:
  • No tiene atributos
Ejemplo:
<!DOCTYPE html>
<html>
<head>
    <title>Título</title>
</head>
<body>
    Contenido
</body>
</html>
Nota: Es obligatorio en todos los documentos HTML5 y debe ser lo primero en el documento.
<html>

Representa la raíz de un documento HTML. Todos los demás elementos deben ser descendientes de este elemento.

Sintaxis:
<html>...</html>
Atributos:
  • lang: Especifica el idioma del documento (ej. "es", "en")
  • xmlns: Define el namespace XML (solo para XHTML)
Ejemplo:
<!DOCTYPE html>
<html lang="es">
<head>
    <title>Título</title>
</head>
<body>
    Contenido
</body>
</html>
Nota: Es el elemento raíz de todo documento HTML y debe incluir los elementos <head> y <body>.
<head>

Contiene metadatos/información sobre el documento, como título, enlaces a hojas de estilo, scripts, etc.

Sintaxis:
<head>...</head>
Atributos:
  • No tiene atributos específicos, pero puede contener varios elementos hijos
Ejemplo:
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página</title>
    <link rel="stylesheet" href="estilos.css">
    <script src="script.js"></script>
</head>
Nota: El contenido del <head> no se muestra directamente en la página, pero es esencial para su funcionamiento y SEO.

Contiene metadatos/información sobre el documento, como título, enlaces a hojas de estilo, scripts, etc.

<body>

Representa el contenido principal del documento HTML que se muestra en el navegador.

Sintaxis:
<body>...</body>
Atributos:
  • onload: Script que se ejecuta cuando la página se carga.
  • onunload: Script que se ejecuta cuando la página se descarga.
Ejemplo:
<body>
        <h1>Bienvenido</h1>
        <p>Este es el contenido principal de la página.</p>
    </body>
Nota: Solo puede haber un elemento <body> en cada documento HTML.
<title>

Define el título del documento que se muestra en la barra de título del navegador o en la pestaña.

Sintaxis:
<title>...</title>
Atributos:
  • No tiene atributos específicos.
Ejemplo:
<head>
        <title>Mi Página Web</title>
    </head>
Nota: Es obligatorio en todos los documentos HTML y debe ser único dentro del elemento <head>.
<meta>

El elemento <meta> define metadatos sobre el documento HTML, como el conjunto de caracteres, la descripción, palabras clave, autor, y más.

Sintaxis:
<meta name="nombre" content="valor">
Atributos:
  • charset: Especifica el conjunto de caracteres del documento (ej. "UTF-8").
  • name: Define el tipo de metadatos (ej. "description", "keywords").
  • content: Proporciona el valor de los metadatos.
  • http-equiv: Proporciona información del encabezado HTTP (ej. "refresh", "content-type").
Ejemplo:
<head>
        <meta charset="UTF-8">
        <meta name="description" content="Esta es una página de ejemplo">
        <meta name="keywords" content="HTML, CSS, JavaScript">
        <meta name="author" content="Tu Nombre">
    </head>
Nota: Los elementos <meta> son esenciales para SEO y para proporcionar información al navegador y motores de búsqueda.
<link>

El elemento <link> define la relación entre el documento actual y un recurso externo, como una hoja de estilo.

Sintaxis:
<link rel="stylesheet" href="estilos.css">
Atributos:
  • rel: Especifica la relación entre el documento y el recurso enlazado (ej. "stylesheet").
  • href: Especifica la URL del recurso enlazado.
  • type: Especifica el tipo MIME del recurso enlazado.
  • media: Especifica el medio para el que está diseñado el recurso enlazado (ej. "screen").
Ejemplo:
<head>
        <link rel="stylesheet" href="estilos.css">
    </head>
Nota: Es comúnmente utilizado para enlazar hojas de estilo CSS externas.
<style>

El elemento <style> se utiliza para definir estilos CSS internos en un documento HTML.

Sintaxis:
<style>
        body {
            background-color: lightblue;
        }
    </style>
Atributos:
  • type: Especifica el tipo MIME del contenido (por defecto "text/css").
  • media: Especifica el medio para el que están diseñados los estilos (ej. "screen").
Ejemplo:
<head>
        <style>
            h1 {
                color: red;
            }
        </style>
    </head>
Nota: Se recomienda usar hojas de estilo externas para mantener el código más limpio.
<script>

El elemento <script> se utiliza para definir o enlazar scripts de JavaScript en un documento HTML.

Sintaxis:
<script src="script.js"></script>
Atributos:
  • src: Especifica la URL del archivo de script externo.
  • type: Especifica el tipo MIME del script (por defecto "text/javascript").
  • async: Indica que el script debe ejecutarse de forma asíncrona.
  • defer: Indica que el script debe ejecutarse después de que el documento haya sido analizado.
Ejemplo:
<head>
        <script src="script.js" defer></script>
    </head>
Nota: Los scripts pueden colocarse en el <head> o al final del <body> para optimizar el rendimiento.
<base>

El elemento <base> especifica la URL base para todos los enlaces relativos en un documento HTML.

Sintaxis:
<base href="https://www.ejemplo.com/">
Atributos:
  • href: Especifica la URL base para los enlaces relativos.
  • target: Especifica el destino predeterminado para los enlaces (ej. "_blank", "_self").
Ejemplo:
<head>
        <base href="https://www.ejemplo.com/" target="_blank">
    </head>
Nota: Solo puede haber un elemento <base> en el documento y debe estar dentro del <head>.

<slot>

El elemento <slot> se utiliza en componentes web para definir puntos de inserción donde se puede colocar contenido personalizado.

Sintaxis:
<slot>...</slot>
Ejemplo:

        </div>
    </template>
    
    <script>
        class MiComponente extends HTMLElement {
            constructor() {
                super();
                const template = document.getElementById('miComponente').content;
                const shadowRoot = this.attachShadow({ mode: 'open' });
                shadowRoot.appendChild(template.cloneNode(true));
            }
        }
        customElements.define('mi-componente', MiComponente);
    </script>

Nota: Es una característica clave de los Web Components para crear interfaces reutilizables.
